# @mui/icons-material 아이콘

SNS UI용 아이콘 barrel. 필요한 아이콘만 import하여 tree-shaking 유지.

## 사용법

```tsx
import { FavoriteIcon, ChatBubbleOutlineIcon, ShareIcon } from '@/icons';

<IconButton><FavoriteIcon /></IconButton>
<IconButton><ChatBubbleOutlineIcon /></IconButton>
<IconButton><ShareIcon /></IconButton>
```

## 카테고리

| 카테고리 | 아이콘 |
|----------|--------|
| 네비게이션 | Home, Menu, Close, ArrowBack, Settings |
| SNS 액션 | Favorite, FavoriteBorder, ChatBubbleOutline, Share, Bookmark |
| 콘텐츠 | Image, Article, AddPhotoAlternate, Edit, Delete |
| 지도 | Place, MyLocation, Straighten |
| 사용자 | Person, PersonOutline, Logout, AdminPanelSettings |
| 검색 | Search, FilterList |

추가 아이콘은 [MUI Icons](https://mui.com/material-ui/material-icons/)에서 확인 후 이 파일에 추가.
