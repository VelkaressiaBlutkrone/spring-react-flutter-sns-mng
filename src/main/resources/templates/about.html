About

서비스 소개

지도 기반 SNS는 사용자 위치를 중심으로 지도 위에 Pin을 붙이고, Pin과 연결된 게시글·이미지 게시글을 공유하는 서비스입니다.

지도 기반 SNS 컨셉

현재 위치 또는 선택한 위치 주변의 정보를 지도에서 한눈에 확인하고, 나만의 Pin을 만들어 관련 콘텐츠를 연결할 수 있습니다.

기술 스택

  • Backend: Spring Boot 4, Java 21, Spring Data JPA
  • Web: Thymeleaf, Bootstrap/Tailwind
  • 지도: Kakao Map / Naver Map / Google Map (추상화)
  • DB: H2(개발), MySQL(운영)
  • 캐시: Redis
